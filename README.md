# webpack-starter
First, install dependencies.
```
npm install
```

## Usage
Use the following entry points for your scripts and styles:
- **src/js/main.js**
- **src/scss/style.scss**

**index.html** is located in the **public** folder. Imports are already set up.


## Run
#### Watch for development
```
npm run watch
```

#### Build for production
```
npm run prod
```
